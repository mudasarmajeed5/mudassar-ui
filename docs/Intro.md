# Intro

@mudassar/ui — TypeScript-first, unstyled React primitives for building accessible components.

Follow on GitHub: https://github.com/mudasarmajeed5

> Note: this is a hobby project. Any components I build will be published here and all components support TypeScript.

Quick start (usage)
```ts
import { Button } from '@mudasarmajeed5/dialog'

<Dialog title="Delete account" onComplete={deleteAccount} >Click me</Dialog>
```

Contributing: PRs and issues welcome — see the repo for guides and examples.
